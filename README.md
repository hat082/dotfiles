# dotfiles
My dotfiles

## nvim
clear cache

```bash
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
```

create a user config

```bash
git clone https://github.com/AstroNvim/user_example.git ~/dotfiles/.config/nvim/lua/user/
rm -rf .git
```
